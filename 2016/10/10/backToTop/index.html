回到顶部效果

发表于 2016-10-10

在浏览淘宝、京东、百度文库等网页时，页面中展示信息量比较多，因此在纵向会产生滚动条，当浏览比较深时，可以通过点击“回到顶部”按钮快速回到页面的顶部。

相关知识点

  1. 获取滚动条距离顶端的距离：

     //IE浏览器：document.documentElement.scrollTop 

//谷歌浏览器：document.body.scrollTop

  2. 获取浏览器可是区域的高度：

    document.documentElement.clientHeight

  3. 事件：

    window.onscroll

  4. 定时器：

    setInterval（code，millisec）；传入2个参数

clearInterval（timer）；传入1个参数

代码

方法一

window.onload=function(){
       var obtn=document.getElementById("btn");
       var timer=null;
       var isTop;

       obtn.onclick=function(){
           timer=setInterval(function(){
               var disTop=document.documentElement.scrollTop || document.body.scrollTop;
               var speed=Math.floor(-disTop/6);
               document.documentElement.scrollTop=document.body.scrollTop=disTop+speed;
               isTop=true;
               if (disTop==0) {
                   clearInterval(timer);
               }
               // console.log(disTop);
           },30)
       }

       window.onscroll=function(){
           var disTop=document.documentElement.scrollTop || document.body.scrollTop;
           var clientHeight=document.documentElement.clientHeight;
           if (disTop>=clientHeight) {
               obtn.style.display="block";
           }else{
               obtn.style.display="none";
           }
           if (!isTop) {
               clearInterval(timer);
           }
           isTop=false;
       }
   }

方法二

window.onload=function(){
      var obtn=document.getElementById('btn');
      var timer;

      obtn.onclick=function(e){
          e.stopPropagation();
          e.preventDefault();
          timer=setInterval(function(){
              var disTop=document.documentElement.scrollTop || document.body.scrollTop;
              var speed=Math.floor(-disTop/5);
              document.documentElement.scrollTop=document.body.scrollTop=disTop+speed;
              if (disTop==0) {
                  clearInterval(timer);
              }
          },30)
      }
      window.onclick=function(){
          clearInterval(timer);
      }
  }

两种方法比较

方法一

点击“回到顶部”按钮时的执行顺序：

  1. 执行setInterval事件，由于有isTop=true，所以当执行下一次scroll的时候，isTop=true；（由于isTop=true，不执行clearInterval()事件，执行完onscroll事件后，istrue=false，再进入setInterval事件，此时isTop=true，所以，在回到顶部之前没有手动触发滚轮，就不会停止向上滚动）
  2. 执行onscroll，先执行（！isTop）判断，然后再给isTop赋值false，重复执行1；
  3. 在回到顶部之前有认为滚动时，重复执行2（此时因为isTop赋值为false，故if（！isTop）为true，即执行clearInterval（）），此时isTop的值为false；

方法二

点击“回到顶部”按钮时的执行顺序：

  1. 执行setInterval事件；
  2. 当有人为单击浏览器时，执行clearInterval();

从方法一和方法二执行的顺序看，方法一在每次执行setInterval()时都要执行onscroll事件，在手动触发滚动条滚动时也会触发onscroll事件，处理机制比较复杂，而方法一没有这些繁琐的操作，处理起来更为简单。
