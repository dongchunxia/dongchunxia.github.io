图片放大镜特效（一）

发表于 2016-07-02

在网购时，在商品的主页上，当我们把鼠标放在图片上时，图片的右边会出现局部放大的图片，鼠标移开，放大的图片消失，这就是图片放大特效。今天在慕课网上看完姜维老师的“用JS实现放大镜特效”课程，现总结如下。

1.技术点

三个鼠标事件：

  1. onmouseover：鼠标移到指定对象上时触发；
  2. onmouseout：鼠标移出指定对象上时触发；
  3. onmousemove：鼠标在指定对象上移动时触发；

六个定位属性：

style.left与offsetLeft的区别

  1. style.left返回的是字符串，如30px，offsetLeft返回的是数值30；
  2. style.left是可读写的，offsetLeft是只读的，所以要改变div的位置，只能修改style.left;

  3. style.left的值需要事先定义，否则取到的值为空。

    2.原理

    放大镜的offsetLeft/(小图片的宽度-放大镜的宽度)=大图片移动的距离/(大图片的宽度-显示区域的宽度)

    3.JavaScript实现

    JavaScript核心代码：

    markBox.onmousemove=function(ev){

      var event= ev ||window.event;

      var left=event.clientX-demo.offsetLeft-smallBox.offsetLeft-float.offsetWidth/2;
      var top=event.clientY-demo.offsetTop-smallBox.offsetTop-float.offsetHeight/2;

      if (left<0) {
          left=0
      }else if (left>smallBox.offsetWidth-float.offsetWidth) {
          left=smallBox.offsetWidth-float.offsetWidth
      }
      if (top<0) {
          top=0;
      }else if (top>smallBox.offsetHeight-float.offsetHeight) {
          top=smallBox.offsetHeight-float.offsetHeight
      }

      float.style.left = left + "px";
      float.style.top= top + "px";

      var percentX=left/(smallBox.offsetWidth-float.offsetWidth);
      var percentY=top/(smallBox.offsetHeight-float.offsetHeight);

      bigImg.style.left= -percentX * (bigImg.offsetWidth-bigBox.offsetWidth)+"px";
      bigImg.style.top= -percentY*(bigImg.offsetHeight-bigBox.offsetHeight)+"px";

4.浏览器兼容

ie中设置对象的透明度：

filter：alpha（opacity=number），number的取值范围是0-100，0代表完全透明；

ie中闪烁情况的解决：
ie中的事件捕获中，当鼠标悬在小图片上时，触发onmouseover事件，鼠标最直接接触的是float，因此紧接着会触发小图片的onmouseout事件，出现闪烁情况。而在谷歌等其他浏览器中不会出现这种情况，解决方法是在小图片上覆盖一层div（id=mark）。

5.总结

该方法的课扩展性比较差，看过大牛用jQuery+Jason写的demo，当图片增多时，只需要给图片加一个属性即可，不需要再单独写代码来获取图片的相关信息。但由于本人能力有限目前还不能礼物其精髓，待技术变强会后续更新。
