基于hexo-theme-next的主题修改

发表于 2016-06-28

前面已经介绍了基于Hexo+Github搭建的个人博客，但是hexo自带的主题不是很喜欢，于是又开始了我的折腾之路。经过查看众多hexo主题Demo，最终还是比较喜欢next和yilia两款主题，下面就以hexo-theme-next主题为例介绍其更改过程。
hexo主题汇总。

安装主题

将Git Bash切换到E:\hexo下，执行下面的命令将next主题下载到themes/hexo-theme-next下。

$ git clone https://github.com/iissnan/hexo-theme-next

修改配置文件

修改你的博客根目录E/Hexo下的_config.yml配置文件中的theme属性，将其设置为hexo-theme-next。

next主题的设置与优化可以参考next的官方文档

更新hexo-theme-next主题

执行如下命令将主题进行更新

$ cd themes/hexo-theme-next
$ git pull
